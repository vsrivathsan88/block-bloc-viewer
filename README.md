# block-bloc-viewer

Public splat viewer for worlds generated by [block-bloc](https://github.com/vsrivathsan88) — sketch → Blender blockout → World Labs Marble.

Loads a `.spz` (Gaussian splat) directly from the World Labs CDN. Default world: `living-room`.

## Usage

`https://vsrivathsan88.github.io/block-bloc-viewer/`

Query params:

- `?spz=<url>` — load any World Labs CDN splat URL (no auth required for public CDN assets).
- `?title=<label>` — override the displayed title.

Example:

```
?spz=https://cdn.marble.worldlabs.ai/<world-id>/<file>_sand_500k.spz&title=My%20Room
```

## How it works

- `@mkkellogg/gaussian-splats-3d` via esm.sh CDN
- Three.js peer dep pinned to 0.158.0 via import map
- Single static HTML — no build step

## Deploy

GitHub Pages serves `index.html` from the repo root.
